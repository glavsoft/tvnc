TightVNC 2.0 Goes Beta

February 23, 2010 – GlavSoft LLC. has released TightVNC 2.0 Beta 1. This is the first public preview of the upcoming TightVNC 2.0 release for Windows systems.

TightVNC is a free software package that shows the desktop of a remote computer and allows to control it with local mouse and keyboard, just like it was a local computer. TightVNC comes with full source code which is GPL-licensed. It's free even for commercial usage (technical support and customization services can be ordered separately).

TightVNC 2.0 Beta 1 delivers completely new TightVNC Server for Windows. It was redesigned and reimplemented from scratch. It features new component-based architecture compatible with the latest operating systems, Windows 7 and Windows 2008 Server R2. Older versions of Windows are supported as well – the minimum requirement is Windows 2000.

New version introduces a number of major improvements, including:

  • new file transfer subsystem with recursive operations, on-the-fly compression and large file support,
  • solid service-mode configuration (no more confusion between "default" and "user" settings),
  • secure control and configuration interface,
  • IP-based access control made easily configurable,
  • new installer, compact and flexible.

TightVNC 2.0 Beta 1 can be downloaded from the TightVNC Web site:

https://www.tightvnc.com/download.php
